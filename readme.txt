== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Upload violet.zip file
3. Click on the 'Activate' button to use your new theme right away.
4. Install 'Violet Custom Post Meta' plugin
5. In order to active map and contact form you nedd to install required plugin (Wp Google Map,Contact Form 7).
5. Navigate to Appearance > Customize and taste as you like.
6. To enable onpage navigation Appearance > Menus then create custom links with section id(#top, #about, #skill, #service, #portfolio, #testimonial, #team, #blog, #contact).
